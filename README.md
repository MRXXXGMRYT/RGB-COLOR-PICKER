# 🎨 RGB Color Mixer
[**Live Demo**](https://mrxxxgmryt.github.io/RGB-COLOR-PICKER/)

A sleek, interactive web-based tool for creating and experimenting with RGB and HEX colors in real-time.

## 🚀 Overview
The **RGB Color Mixer** is a responsive application designed for developers and designers. It allows you to visualize colors instantly by adjusting sliders, typing manual values, or selecting from a range of presets.

## ✨ Features
- **Real-time Color Mixing:** Watch the entire page background transform as you adjust values.
- **Dual Input Control:** - **Sliders:** Smoothly scroll through the 0-255 range.
  - **Typeable Fields:** Click and type exact numbers for precision.
- **Quick Copy:** Dedicated buttons to copy `rgb()` or `HEX` codes to your clipboard instantly.
- **Color Presets:** A palette of standard colors for quick selection.
- **Keyboard Shortcuts:**
  - `R` — Generate a random color.
  - `C` — Copy RGB code.
  - `H` — Copy HEX code.
  - `Esc` — Reset to default state.
- **Responsive & Modern:** Built with **Tailwind CSS** and **Font Awesome** for a clean, mobile-friendly UI.

## 🛠️ Built With
- **HTML5**
- **Tailwind CSS** (via CDN)
- **JavaScript** (Vanilla)
- **Font Awesome** (Icons)
